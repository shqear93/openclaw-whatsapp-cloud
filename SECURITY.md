# Security

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities — this includes anything related to:

- Webhook signature verification (`src/signature.ts`)
- Credential or token handling
- Access-control / allowlist bypass

Instead, use [GitHub's private vulnerability reporting](https://github.com/shqear93/openclaw-whatsapp-cloud/security/advisories/new) for this repository, or email the maintainer directly (see the repository owner's GitHub profile).

## Supported versions

Only the latest published version is supported. Please upgrade before reporting an issue.
